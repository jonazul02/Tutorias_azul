# Tutorias Datascience
Este repositorio contiene los notebooks desarrollados durante las tutorías. 
Cada colaborador trabajará en su rama `feature_nombre_personal` y los cambios se fusionarán en la rama `main` después de una revisión.

## Estructura del Repositorio:
- `Tutorias_DS`: Carpeta donde se almacenan las presentaciones e información necesaria para compartir.
- `README.md`: Información básica del repositorio.
- `.gitignore`: Lista de archivos a excluir del control de versiones.

## Instrucciones para Colaboradores:
1. Crea una nueva rama `feature_nombre_personal` desde `main`.
2. Haz tus cambios en la nueva rama.
3. Abre un pull request cuando tus cambios estén listos para revisión.
